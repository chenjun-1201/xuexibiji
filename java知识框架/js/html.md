# html

- div 和span 都是盒子模型,**div是一个大盒子模型，能独占一行**

```html
<body>
    <div id="k">这是一行div</div>
    <div id="k">这是一行div</div>
    <div id="k">这是一行div</div>
    <span>span</span>
    <span>span</span>
    <span>span</span>
  区别span是一个小盒子模型，div是一个大盒子模型，能独占一行，span不能占一行
</body>
```
